# Roamly – AI Based Travel Planner 🌍🤖

Roamly is an AI-powered travel planning application that generates personalized travel itineraries based on user preferences such as destination, budget, travel duration, and interests.

## 🚀 Features
- AI-based trip itinerary generation
- Personalized travel plans
- Budget-friendly recommendations
- Interactive UI
- Real-time responses

## 🛠 Tech Stack
- Frontend: React / React Native
- Backend: Node.js / Express
- Database: MongoDB / Firebase
- AI API: Gemini / OpenAI
- Tools: GitHub, Postman
